# Design Point - Next.js Landing

This is a minimal Next.js (App Router) scaffold with Tailwind CSS for the Design Point landing page.

Theme: white + light blue

Quick start (PowerShell):

```powershell
# 1) Install dependencies
npm install

# 2) Run dev server
npm run dev

# 3) Open in browser
# http://localhost:3000
```

Notes:
- The homepage is implemented at `app/page.tsx` and is a client component (uses `use client`).
- The site uses `framer-motion` and `lucide-react` for icons and simple animations.
- Tailwind is configured in `tailwind.config.js` and styles in `styles/globals.css`.
